# Coral Behavior & Health Monitor Streamlit Dashboard

This directory houses the interactive Streamlit dashboard presenting your screen behaviors, mobile steps, sleep, and Coral SQL query execution.

---

## 🛠️ How to Run Locally

### 1. Activate Virtual Environment
Open your shell and activate the python virtual environment:
```powershell
.venv\Scripts\Activate.ps1
```

### 2. Install Dependencies
Install Streamlit and pandas inside the activated environment:
```bash
pip install streamlit pandas
```

### 3. Launch App
Run the Streamlit application:
```bash
streamlit run app/app.py
```
*The app will automatically open in your default browser at `http://localhost:8501`!*
